Задание: "Простое хранилище"
Создай класс Storage с использованием generics, который может хранить один предмет любого типа.
Требования:

Класс должен называться Storage и использовать generics
Внутри должна быть переменная для хранения одного предмета
Методы:

store(item) - сохранить предмет
get() - получить предмет (может вернуть null если ничего не сохранено)
isEmpty() - проверить пустое ли хранилище



В main() создай:

Хранилище для строк, положи туда "Hello", выведи содержимое
Хранилище для чисел, положи туда 42, выведи содержимое
Покажи что хранилища работают независимо друг от друга
