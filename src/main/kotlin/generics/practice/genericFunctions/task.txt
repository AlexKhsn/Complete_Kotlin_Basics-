Задание 2: "Generic функции"
Создай обычные функции (не в классе) с использованием generics:
Требования:

Функция swap<T>(a: T, b: T) - которая меняет местами два значения и выводит результат
Функция isEqual<T>(first: T, second: T): Boolean - сравнивает два значения любого типа
Функция getFirst<T>(items: List<T>): T? - возвращает первый элемент списка любого типа

В main() протестируй:

swap с двумя строками
swap с двумя числами
isEqual с одинаковыми строками
isEqual с разными числами
getFirst со списком строк
getFirst с пустым списком чисел

Подсказка по синтаксису: в функциях <T> пишется перед именем функции.
Попробуй написать код сам. Если застрянешь или сделаешь - показывай результат.
