Задание 1: "Транспорт"
Создай базовый класс Vehicle (транспортное средство), который имеет:

Свойство brand (марка) типа String
Свойство maxSpeed (максимальная скорость) типа Int
Метод displayInfo(), который выводит информацию о транспорте в формате:
"Транспорт [марка] с максимальной скоростью [скорость] км/ч"

Создай класс Car (автомобиль), который наследуется от Vehicle и имеет:

Собственные значения для марки и скорости
Переопределенный метод displayInfo(), который выводит информацию об автомобиле,
а затем вызывает родительский метод

Создай объект класса Car и вызови его метод displayInfo().
Покажи код и результат выполнения.
