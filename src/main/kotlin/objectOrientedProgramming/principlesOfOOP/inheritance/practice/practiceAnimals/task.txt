Задание 2: "Животные"
Создай базовый класс Animal (животное), который имеет:

Свойство name (имя) типа String
Свойство age (возраст) типа Int
Метод makeSound(), который выводит "Животное издает звук"
Метод introduce(), который выводит "Меня зовут [имя], мне [возраст] лет"

Создай класс Dog (собака), который наследуется от Animal и:

Принимает name и age через конструктор
Переопределяет метод makeSound(), чтобы он выводил "Гав-гав!"
Добавляет собственный метод wagTail(), который выводит "[имя] виляет хвостом"

Создай класс Cat (кот), который наследуется от Animal и:

Принимает name и age через конструктор
Переопределяет метод makeSound(), чтобы он выводил "Мяу!"
Добавляет собственный метод purr(), который выводит "[имя] мурчит"

Создай объекты собаки и кота, вызови их методы.
Покажи код и результат выполнения.
