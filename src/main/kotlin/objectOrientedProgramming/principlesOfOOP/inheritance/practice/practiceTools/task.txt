Задание: "Инструменты"
Создай базовый класс Tool (инструмент), который имеет:

Свойство name (название инструмента)
Свойство material (материал, из которого сделан)
Метод use(), который выводит "Использую [название] из [материал]"

Создай класс Hammer (молоток), который наследуется от Tool и:

Имеет собственные значения для названия и материала
Переопределяет метод use(), который:

Выводит "Забиваю гвозди молотком из [материал]"
Затем вызывает родительский метод use()



Создай объект класса Hammer и вызови его метод use().
Ожидаемый вывод должен быть примерно таким:
Забиваю гвозди молотком из стали
Использую молоток из стали
Покажи код и результат выполнения.
