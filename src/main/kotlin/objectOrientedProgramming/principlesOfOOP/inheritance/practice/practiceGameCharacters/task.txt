Задание 5: "Игровые персонажи"
Создай абстрактную систему игровых персонажей:
Создай базовый класс Character (персонаж), который имеет:

Свойства name (имя), health (здоровье), level (уровень)
Метод introduce(), который выводит базовую информацию о персонаже
Абстрактный метод attack(), который возвращает урон (Int)
Метод takeDamage(damage: Int), который уменьшает здоровье

Создай класс Warrior (воин), который:

Добавляет свойство weaponDamage (урон оружия)
Переопределяет attack() - возвращает weaponDamage * level
Переопределяет introduce() - добавляет информацию о том, что это воин

Создай класс Mage (маг), который:

Добавляет свойство spellPower (сила заклинаний)
Переопределяет attack() - возвращает spellPower + (level * 2)
Добавляет метод heal(target: Character, amount: Int),
который восстанавливает здоровье другому персонажу

Создай двух персонажей, проведи между ними бой (атака/лечение),
покажи изменения их характеристик.
Покажи код и результат выполнения.
