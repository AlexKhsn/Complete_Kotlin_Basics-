Задание 4: "Сотрудники компании"
Создай базовый класс Employee (сотрудник), который имеет:

Свойства name (имя), id (ID сотрудника),
baseSalary (базовая зарплата)
Метод getDetails(), который возвращает строку с
информацией о сотруднике
Виртуальный метод calculateSalary(), который
возвращает базовую зарплату

Создай класс Manager (менеджер), который
наследуется от Employee и:

Добавляет свойство teamSize (размер команды)
Переопределяет calculateSalary() - добавляет
бонус 1000 руб. за каждого члена команды
Переопределяет getDetails() - добавляет информацию
о размере команды

Создай класс Developer (разработчик), который
наследуется от Employee и:

Добавляет свойство programmingLanguage
(язык программирования)
Переопределяет calculateSalary() - добавляет
бонус 5000 руб. за знание языка
Добавляет метод writeCode(), который выводит
"[имя] пишет код на [язык]"

Создай объекты менеджера и разработчика,
выведи их детали и зарплаты.
Покажи код и результат выполнения.
