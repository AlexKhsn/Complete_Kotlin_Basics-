Задание: "Банковская система"
Создай трёхуровневую иерархию классов для банковской системы:
Уровень 1: Базовый класс Account (счёт), который имеет:

Свойства: accountNumber (номер счёта), ownerName (имя владельца), balance (баланс)
Метод deposit(amount: Double) - пополнение счёта
Метод getAccountInfo() - информация о счёте
Метод withdraw(amount: Double): Boolean - снятие денег (возвращает успех/неудачу)

Уровень 2: Класс SavingsAccount (сберегательный счёт), который:

Добавляет свойство interestRate (процентная ставка)
Переопределяет withdraw() - нельзя снимать больше баланса
Добавляет метод calculateInterest() - рассчитывает проценты

Уровень 3: Класс PremiumSavingsAccount (премиум сберегательный), который:

Добавляет свойство minimumBalance (минимальный баланс)
Переопределяет withdraw() - нельзя снимать ниже минимального баланса
Переопределяет calculateInterest() - повышенная ставка для премиум счетов

Дополнительно:

Создай список List<Account> с разными типами счетов
Напиши функцию processAccounts(accounts: List<Account>,
operation: (Account) -> Unit) которая применяет операцию ко всем счетам
Обработай исключения при некорректных операциях

Программа должна:

Создать счета всех типов
Добавить их в общий список
Выполнить операции пополнения/снятия
Показать информацию обо всех счетах через функцию высшего порядка

Покажи код и результат выполнения.
